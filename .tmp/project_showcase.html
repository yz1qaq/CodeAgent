🧩 核心模块

📦 Agent 模块 (agent/)

核心 Agent 实现，基于 LangChain 的 create_agent 创建智能代理。支持多轮对话、工具调用和记忆功能。

code_agent.py

📦 模型模块 (model/)

配置和管理 LLM 模型，当前使用阿里云 DashScope 的 Qwen3.5-Plus 模型。

model.py

📦 工具模块 (tools/)

提供 Agent 可调用的各种工具，包括文件管理、命令行执行和终端控制功能。

file_tools.pycli_tools.pyterminal_tools.py

📦 MCP 服务模块 (m_mcp/)

基于 MCP (Model Context Protocol) 标准实现的各种工具服务，支持 stdio 传输方式。

cli_tools.pyterminal_tools.pypowershell_tools.py

📦 工具模块 (utils/)

通用工具函数，包括 MCP 客户端创建等辅助功能。

mcp.py

📦 提示词模块 (prompts/)

定义 Agent 的系统提示词和对话模板。

prompts.py

🔧 核心功能

📝 文件管理工具

通过 LangChain 的 FileManagementToolkit 实现，支持在指定目录进行文件操作。

💻 命令行工具

通过 MCP 服务封装命令行执行功能，支持安全的命令执行（禁止 rm 命令）。

🖥️ 终端控制工具

通过 AppleScript 控制 macOS 终端，支持打开/关闭终端、执行命令、发送键盘输入等。

open_new_terminalclose_terminal_if_openrun_script_in_exist_terminalsend_terminal_keyboard_keyget_terminal_full_text

💾 持久化存储

支持 MongoDB 和 Redis 两种检查点存储方式，实现对话历史的持久化。
